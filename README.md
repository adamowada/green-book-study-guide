# Memorize the Green Book

A local-first study tool for memorizing selected sections of the Army's Green Book:

- Army Values
- The Soldier's Creed
- Military time
- General orders
- Special orders
- Phonetic alphabet
- Army rank structure, with images

The planned app is a single-page Next.js website with Tailwind styling and browser local storage only. It will provide easy and hard fill-in-the-blank modes, grading with corrections, clear-all, submit, and retake actions.

## Source Material

The study material is based on `TP600-4 The Soldiers Green Book_Aug 2025.pdf`, included in this repository.
